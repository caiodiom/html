O que é HTML Semântico?

(Voltar)

HTML semântico significa usar tags que têm significado, ou seja, que dizem ao navegador e aos leitores (inclusive leitores de tela) qual é o papel daquele conteúdo. Isso ajuda na acessibilidade, SEO e na organização do código. Por exemplo, usar <article> para uma notícia é mais apropriado do que usar apenas uma <div>.

HTML Semântico com Tags Básicas

(Voltar)

HTML semântico é uma forma de escrever o código HTML usando tags com significado, que ajudam a organizar melhor o conteúdo da página, melhorando a acessibilidade e o SEO. Um bom exemplo é usar a tag <header> para definir o topo do site. Evite usar apenas <div> para tudo, prefira tags específicas sempre que possível.

Tags Semânticas Comuns

(Voltar)
Tag Função
<header> Define o cabeçalho da página ou de uma seção. Pode conter título, logo e menus.
<nav> Cria uma área de navegação (menus). Exemplo: Menu principal
<main> Delimita o conteúdo principal, como artigos, seções e informações relevantes.
<section> Seções temáticas dentro do conteúdo. Por exemplo: Sobre nós, Serviços.
<article> Conteúdo independente, como postagens ou notícias.
"Este artigo é um exemplo."
<aside> Conteúdo complementar (ex: barra lateral com links ou informações extras).
<footer> Rodapé da página, geralmente contém direitos autorais, contatos ou links adicionais.
